## Descripción del Proyecto

Sonriente es una landing page profesional diseñada para una clínica odontológica ubicada en Santo Domingo, Ecuador. La página presenta la identidad de la marca, comunica los servicios odontológicos ofrecidos y facilita la reserva de citas a través de un formulario de contacto accesible y funcional.

La página está estructurada para generar confianza en los pacientes potenciales mediante la exhibición de instalaciones, testimonios reales, indicadores de experiencia y un mensaje claro: **"Sonrisas que transforman vidas"**.
